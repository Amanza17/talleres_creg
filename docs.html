Volver

Objetivo

Escribe un programa Python que controle el coche durante una vuelta completa. El coche tiene tres sensores frontales que detectan la linea negra. Si se sale demasiados frames, la simulacion termina.

API Disponible

izq, centro, der = leer_linea()

Devuelve 1 si cada sensor ve la linea negra y 0 si ve asfalto.

error = leer_error()

Devuelve un error aproximado: negativo hacia izquierda, positivo hacia derecha.

motor(izquierda, derecha)

Potencia de cada lado entre -100 y 100. Diferencia de motores = giro.

esperar()

Avanza un frame de simulacion. Si no llamas a esta funcion, el coche no se mueve.

leer_velocidad()

Velocidad actual. Util para limitar rectas y curvas.

leer_progreso()

Porcentaje de vuelta completada entre 0 y 1.

tiempo()

Numero de frames simulados.

imprimir(...)

Escribe en consola sin saturar el navegador.

Plan de Trabajo

  1. Haz que calcular_error() convierta sensores en un numero.
  2. Guarda memoria del ultimo lado donde viste la linea para recuperarte si los tres sensores leen 0.
  3. Implementa un controlador P: correccion = error * KP.
  4. Anade derivada: usa error - ultimo_error para suavizar cambios bruscos.
  5. Baja velocidad en curva: si abs(error) sube, usa VEL_CURVA.
  6. Depura con imprimir() cada 50 pasos, no en cada frame.

Consejos

  • Si vibra en recta, baja KP o anade KD.
  • Si se abre en curvas, baja velocidad antes de subir mucho KP.
  • Si pierde la linea en una horquilla, usa memoria: gira hacia el ultimo lado visto.
